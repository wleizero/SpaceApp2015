h1. Tangleball Hack Sessions SpaceSim

This app lets people more clearly see what is happening and being checked during a orbital launch process.


h2. Ruby on Rails

This application requires:

* Ruby version 2.0.0
* Rails version 4.0.0

Learn more about "Installing Rails":http://railsapps.github.io/installing-rails.html.

h2. Development

* Template Engine: Haml
* Testing Framework: Test::Unit
* Front-end Framework: Twitter Bootstrap (Sass)
* Form Builder: SimpleForm
* Authentication: None
* Authorization: None
